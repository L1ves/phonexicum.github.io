# Pentest linux distributions

---

[Distro Tools List](http://www.kitploit.com/search/label/Distro#) - list of security linux destributives
<br> [Лучшие дистрибутивы для проведения тестирования на проникновение](https://habrahabr.ru/company/pentestit/blog/276477/)

* [kali linux](https://www.kali.org/) (+ exists kali for android - *NetHunter*) - penetration testing and ethical hacking linux distribution (xfce - looks like the most lightweight)
* [ptf](https://github.com/trustedsec/ptf) - The PenTesters Framework (PTF) - a way for modular support for up-to-date tools
* Other pentest distros:

    * [BlackArch](https://blackarch.org/tools.html) - concept: *contain everything*
    * [Parrot Security OS](https://www.parrotsec.org/download.fx) - good kali alternative
    * [Backbox](https://backbox.org/) (ubuntu-based)
    * [Devuan](https://devuan.org/), [pentoo](http://www.pentoo.ch/download/), [bugtraq](http://bugtraq-team.com/downloads.html), [Cyborg](http://cyborg.ztrela.com/), [Matriux](https://sourceforge.net/projects/matriux/), [ArchAssault](https://sourceforge.net/projects/archassault/)
    * [samurai](https://sourceforge.net/projects/samurai/) - targeted at web-pentest
    * [NST (Network Security Toolkit)](https://sourceforge.net/projects/nst/)
    * [Wifislax](http://www.wifislax.com/); [airslax](http://airslax.ru/); [Xiaopan OS](https://sourceforge.net/projects/xiaopanos/) - wifi security
    * [Santoku linux](https://santoku-linux.com/); [Android Tamer](https://androidtamer.com/) - targeted at mobile security

* [DEFT linux](http://www.deftlinux.net/) - a linux distribution for computer forensics
* [pentestbox](https://pentestbox.org/#download) - collection of hacking tools for windows

<br>

* [osboxes.org](https://www.osboxes.org/) - prebuilt OS
* [tb.rg-adguard.net](https://tb.rg-adguard.net/public.php) - microsoft products

*Caution!* - hackers hack other hackers - divide your personal accounts/environment and your working pentest env. <br>
*Caution!* - Kali linux, etc. has a lots of built-in tools, which is not really thoroughly checked for vulnerabilities, therefore all hacking destributions is highly dangerous.

# Methodologies

---

[**Penetration testing methodologies** (owasp)](https://www.owasp.org/index.php/Penetration_testing_methodologies):

* PTES - [Penetration Testing Execution Standard](http://www.pentest-standard.org/index.php/Main_Page) <small>*(похожа на ISO 27001)*</small>
* OSSTMM - [Open Source Security Testing Methodology Manual](http://www.isecom.org/research/osstmm.html)
    <br> Formalized and structured document. It includes security testing, security analysis, operational security metrics, trust analysis, operational trust metrics, the Möbius Defense, and the essential tactics for testing the security of anything. Contains detailed, stage-by-stage description of penetration testing process.
* ISSAF - [Information Systems Security Assessment Framework](http://www.oissg.org/issaf)
* NIST SP 800-115 - [Technical Guide to Information Security Testing and Assessment (2008)](https://nvlpubs.nist.gov/nistpubs/Legacy/SP/nistspecialpublication800-115.pdf)
* [ISO/IEC 27000-series](https://en.wikipedia.org/wiki/ISO/IEC_27000-series#Published_standards)
    <br> [Государственные стандарты РФ](https://ru.wikipedia.org/wiki/%D0%A1%D1%82%D0%B0%D0%BD%D0%B4%D0%B0%D1%80%D1%82%D1%8B_%D0%B8%D0%BD%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%86%D0%B8%D0%BE%D0%BD%D0%BD%D0%BE%D0%B9_%D0%B1%D0%B5%D0%B7%D0%BE%D0%BF%D0%B0%D1%81%D0%BD%D0%BE%D1%81%D1%82%D0%B8#.D0.93.D0.BE.D1.81.D1.83.D0.B4.D0.B0.D1.80.D1.81.D1.82.D0.B2.D0.B5.D0.BD.D0.BD.D1.8B.D0.B5_.28.D0.BD.D0.B0.D1.86.D0.B8.D0.BE.D0.BD.D0.B0.D0.BB.D1.8C.D0.BD.D1.8B.D0.B5.29_.D1.81.D1.82.D0.B0.D0.BD.D0.B4.D0.B0.D1.80.D1.82.D1.8B_.D0.A0.D0.A4)
    <br> ГОСТ Р ИСО МЭК 15408-1-2012 / ГОСТ Р ИСО МЭК 15408-2-2013 / ГОСТ Р ИСО МЭК 15408-3-2013
